# hardware-ecommerce-js
Este proyecto es la entrega final correspondiente al curso certificado de Front-End JS del programa Talento Tech. Consiste en el desarrollo de un e-commerce funcional para la venta de componentes de hardware, enfocado en la manipulación dinámica del DOM y la gestión de estados.

[🌐 Haz clic aquí para ver el proyecto](https://alexisrodriguezar.github.io/hardware-ecommerce-js/)

## 🚀 Descripción del Proyecto
La aplicación permite a los usuarios visualizar un catálogo de productos, gestionar un carrito de compras y persistir la información de manera local. El proyecto fue diseñado priorizando la modularidad del código y una experiencia de usuario fluida y responsiva.

## 🛠️ Tecnologías Utilizadas
* **HTML5 & CSS3**: Estructura semántica y diseño responsivo adaptado a múltiples dispositivos (Mobile-first).

* **JavaScript (Vanilla)**: Lógica de negocio, manipulación del DOM y gestión de eventos.

* **Fetch API**: Consumo de datos desde un archivo JSON local para el renderizado dinámico de productos.

* **LocalStorage**: Persistencia del carrito de compras para que el usuario pueda retomar su sesión sin perder sus artículos.

* **Git & GitHub**: Control de versiones del flujo de desarrollo.

## ⚙️ Características Principales

* **Catálogo Dinámico**: Renderizado de productos a partir de un archivo JSON.

* **Carrito de Compras**: Funcionalidades completas de agregar, eliminar productos y vaciar el carrito.

* **Contador en tiempo real**: Actualización automática del stock/cantidad de artículos en el header.

* **Diseño Responsivo**: Implementación de Flexbox, CSS Grid y Media Queries para garantizar la visualización óptima en móviles, tablets y escritorio.
